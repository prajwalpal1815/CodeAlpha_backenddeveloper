# Event Registration System

A Django REST Framework based Event Registration System developed as part of the CodeAlpha Backend Development Internship.

## Features

* Create and manage events
* Register users for events
* Django Admin Panel
* REST API endpoints
* SQLite Database
* Live deployment on Render

## Tech Stack

* Python
* Django
* Django REST Framework
* SQLite
* Git & GitHub
* Render

## API Endpoints

### Event List

`/api/events/`

Returns all available events.

### Event Details

`/api/events/<id>/`

Returns details of a specific event.

### Register Event

`/api/register/`

Allows users to register for an event.
